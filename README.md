# WoW Zone Generator

Used to generate random World of Warcraft zone names for art prompts.

## Getting Started

```
git clone https://github.com/insuusvenerati/user-dashboard-react

yarn

yarn start
```

### Prerequisites

- Latest NodeJS

- Latest Yarn
